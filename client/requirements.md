## Packages
recharts | Dashboard analytics charts (Pie/Bar charts)
date-fns | Date formatting for application and interview dates
framer-motion | Smooth transitions and layout animations for Kanban board
clsx | Utility for conditional class names (standard in shadcn)
tailwind-merge | Utility for merging tailwind classes (standard in shadcn)
react-day-picker | Calendar component date selection

## Notes
Authentication uses Replit Auth (already setup in backend/hooks).
Dashboard requires aggregating data from applications table.
Kanban board needs drag-and-drop or column layout for statuses.
Status colors mapping: Saved (gray), Applied (blue), Interview (purple), Offer (green), Rejected (red).
